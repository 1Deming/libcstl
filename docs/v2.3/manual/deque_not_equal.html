2.2.35. deque_not_equal

  • 函数描述

测试两个双端队列容器是否不等。

  • 函数原型
bool_t deque_not_equal(const deque_t* cpdeq_first, const deque_t* cpdeq_second);
  • 参数说明
cpdeq_first: 第一个双端队列容器。
cpdeq_second: 第二个双端队列容器。
  • 备注

cpdeq_first和cpdeq_second不相等返回true，否则返回false。 cpdeq_first和cpdeq_second必须有效的双端队列容器，否则函数行为未定义。 cpdeq_first和cpdeq_second必须包含相同的数据类型，则函数的行为是未定义的。

  • 前提条件

包含头文件<cstl/cdeque.h>

  • 示例代码
/*
 * deque_not_equal.c
 */

#include <stdio.h>
#include <cstl/cdeque.h>

int main(int argc, char* argv[])
{
    deque_t* pdeq_q1 = NULL;
    deque_t* pdeq_q2 = NULL;
    deque_t* pdeq_q3 = NULL;

    if ((pdeq_q1 = create_deque(int)) == NULL) {
        return -1;
    }
    if ((pdeq_q2 = create_deque(int)) == NULL) {
        deque_destroy(pdeq_q1);
        return -1;
    }
    if ((pdeq_q3 = create_deque(int)) == NULL) {
        deque_destroy(pdeq_q1);
        deque_destroy(pdeq_q2);
        return -1;
    }

    deque_init(pdeq_q1);
    deque_init(pdeq_q2);
    deque_init_n(pdeq_q3, 10);

    if (deque_not_equal(pdeq_q1, pdeq_q2)) {
        printf("q1 != q2\n");
    } else {
        printf("q1 == q2\n");
    }

    if (deque_not_equal(pdeq_q1, pdeq_q3)) {
        printf("q1 != q3\n");
    } else {
        printf("q1 == q3\n");
    }

    deque_destroy(pdeq_q1);
    deque_destroy(pdeq_q2);
    deque_destroy(pdeq_q3);

    return 0;
}
  • 结果输出

q1 == q2
q1 != q3
