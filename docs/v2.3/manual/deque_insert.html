2.2.28. deque_insert

  • 函数描述

向双端队列容器中插入一个数据，并且返回指向这个数据的迭代器。

  • 函数原型
deque_iterator_t deque_insert(deque_t* pdeq_deque, deque_iterator_t it_pos, elem);
  • 参数说明
pdeq_deque: 双端队列容器。
it_pos: 插入数据的位置。
elem: 被插入的数据。
  • 备注

pdeq_deque必须有效的且已经初始化的双端队列容器，否则函数的行为是未定义的。 it_pos必须是属于pdeq_deque的有效的迭代器，否则函数的行为是未定义的。 elem必须和pdeq_deque中的数据类型一致，否则函数的行为是未定义的。

  • 前提条件

包含头文件<cstl/cdeque.h>

  • 示例代码
/*
 * deque_insert.c
 */

#include <stdio.h>
#include <cstl/cdeque.h>

int main(int argc, char* argv[])
{
    deque_t*            pdeq = create_deque(int);
    deque_iterator_t    it;

    if  (pdeq == NULL) {
        return -1;
    }

    deque_init(pdeq);

    deque_push_back(pdeq, 10);
    deque_push_back(pdeq, 20);
    deque_push_back(pdeq, 30);

    printf("pdeq = ");
    for (it = deque_begin(pdeq);
         !iterator_equal(it, deque_end(pdeq));
         it = iterator_next(it)) {
        printf("%d ", *(int*)iterator_get_pointer(it));
    }
    printf("\n");

    it = deque_insert(pdeq, iterator_next(deque_begin(pdeq)), 100);
    printf("The inserted element is %d\n", *(int*)iterator_get_pointer(it));

    printf("pdeq = ");
    for (it = deque_begin(pdeq);
         !iterator_equal(it, deque_end(pdeq));
         it = iterator_next(it)) {
        printf("%d ", *(int*)iterator_get_pointer(it));
    }
    printf("\n");

    deque_destroy(pdeq);

    return 0;
}
  • 结果输出

pdeq = 10 20 30
The inserted element is 100
pdeq = 10 100 20 30
