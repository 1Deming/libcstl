2.2.24. deque_init_copy_array

  • 函数描述

使用数组初始化双端队列容器。

  • 函数原型
void deque_init_copy_array(deque_t* pdeq_dest, const void* cpv_array, size_t t_count);
  • 参数说明
pdeq_dest: 被初始化的双端队列容器。
cpv_array: 用于初始化的数组。
t_count: 数组元素个数。
  • 备注

pdeq_dest必须是未初始化的双端队列容器，否则函数的行为是未定义的。 cpv_array必须是有效的数组，否则函数的行为是未定义的。 pdeq_dest和cpv_array中包含的数据类型必须相同，否则函数的行为是未定义的。

  • 前提条件

包含头文件<cstl/cdeque.h>

  • 示例代码
/*
 * deque_init_copy_array.c
 */

#include <stdio.h>
#include <cstl/cdeque.h>

int main(int argc, char* argv[])
{
    deque_t* pdeq = create_deque(int);
    int      an_array[] = {8, 55, -23, 4444, 1024};
    int      i = 0;

    if (pdeq == NULL) {
        return -1;
    }

    deque_init_copy_array(pdeq, an_array, 5);
    printf("The size of pdeq is %d.\n", deque_size(pdeq));
    printf("The pdeq = ");
    for (i = 0; i < deque_size(pdeq); ++i) {
        printf("%d ", *(int*)deque_at(pdeq, i));
    }
    printf("\n");

    deque_destroy(pdeq);

    return 0;
}
  • 结果输出

The size of pdeq is 5.
The pdeq = 8 55 -23 4444 1024
