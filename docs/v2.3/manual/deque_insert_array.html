2.2.29. deque_insert_array

  • 函数描述

向双端队列容器中插入一个数组。

  • 函数原型
void deque_insert_array(
    deque_t* pdeq_deque, deque_iterator_t it_pos, const void* cpv_array, size_t t_count);
  • 参数说明
pdeq_deque: 双端队列容器。
it_pos: 插入数据的位置。
cpv_array: 被插入的数组。
t_count: 数组中数据的个数。
  • 备注

pdeq_deque必须有效的且已经初始化的双端队列容器，否则函数的行为是未定义的。 it_pos必须是属于pdeq_deque的有效的迭代器，否则函数的行为是未定义的。 cpv_array中的数据和pdeq_deque中的数据类型一致，否则函数的行为是未定义的。

  • 前提条件

包含头文件<cstl/cdeque.h>

  • 示例代码
/*
 * deque_insert_array.c
 */

#include <stdio.h>
#include <cstl/cdeque.h>

int main(int argc, char* argv[])
{
    deque_t*            pdeq = create_deque(int);
    deque_iterator_t    it;
    int                 an_array[] = {-4, -9, -130, -888, -2};

    if  (pdeq == NULL) {
        return -1;
    }

    deque_init(pdeq);
    deque_push_back(pdeq, 10);
    deque_push_back(pdeq, 20);
    deque_push_back(pdeq, 30);

    printf("pdeq = ");
    for (it = deque_begin(pdeq);
         !iterator_equal(it, deque_end(pdeq));
         it = iterator_next(it)) {
        printf("%d ", *(int*)iterator_get_pointer(it));
    }
    printf("\n");

    deque_insert_array(pdeq, iterator_next(deque_begin(pdeq)), an_array, 5);

    printf("pdeq = ");
    for (it = deque_begin(pdeq);
         !iterator_equal(it, deque_end(pdeq));
         it = iterator_next(it)) {
        printf("%d ", *(int*)iterator_get_pointer(it));
    }
    printf("\n");

    deque_destroy(pdeq);

    return 0;
}
  • 结果输出

pdeq = 10 20 30
pdeq = 10 -4 -9 -130 -888 -2 20 30
