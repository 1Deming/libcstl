2.2.26. deque_init_elem

  • 函数描述

使用指定的数据初始化双端队列容器。

  • 函数原型
void deque_init_elem(deque_t* pdeq_deque, size_t t_count, elem);
  • 参数说明
pdeq_deque: 被初始化的双端队列容器。
t_count: 数据个数。
elem: 指定的数据。
  • 备注

pdeq_dest必须是未初始化的双端队列容器，否则函数的行为是未定义的。 pdeq_dest中包含的数据类型必须和elem相同，否则函数的行为是未定义的。

  • 前提条件

包含头文件<cstl/cdeque.h>

  • 示例代码
/*
 * deque_init_elem.c
 */

#include <stdio.h>
#include <cstl/cdeque.h>

int main(int argc, char* argv[])
{
    deque_t* pdeq = create_deque(int);
    int      i = 0;

    if (pdeq == NULL) {
        return -1;
    }

    deque_init_elem(pdeq, 10, 999);
    printf("The size of pdeq is %d\n", deque_size(pdeq));
    printf("The pdeq is ");
    for (i = 0; i < deque_size(pdeq); ++i) {
        printf("%d ", *(int*)deque_at(pdeq, i));
    }
    printf("\n");

    deque_destroy(pdeq);

    return 0;
}
  • 结果输出

The size of pdeq is 10
The pdeq is 999 999 999 999 999 999 999 999 999 999
