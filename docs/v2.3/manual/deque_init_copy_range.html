2.2.25. deque_init_copy_range

  • 函数描述

使用指定范围的数据初始化双端队列容器。

  • 函数原型
void deque_init_copy_range(deque_t* pdeq_dest, iterator_t it_begin, iterator_t it_end);
  • 参数说明
pdeq_dest: 被初始化的双端队列容器。
it_begin: 数据区间的开始位置迭代器。
it_end: 数据区间的末尾位置迭代器。
  • 备注

pdeq_dest必须是未初始化的双端队列容器，否则函数的行为是未定义的。 [it_begin, it_end)必须是有效的数据区间，否则函数的行为是未定义的。 pdeq_dest和[it_begin, it_end)中包含的数据类型必须相同，否则函数的行为是未定义的。

  • 前提条件

包含头文件<cstl/cdeque.h>

  • 示例代码
/*
 * deque_init_copy_range.c
 */

#include <stdio.h>
#include <cstl/cdeque.h>
#include <cstl/cvector.h>

int main(int argc, char* argv[])
{
    deque_t*  pdeq = NULL;
    vector_t* pvec = NULL;
    int       an_array[] = {8, 0, 127, 909, 111, -174, 1233, 0, 65, 28};
    int       i = 0;

    pdeq = create_deque(int);
    if (pdeq == NULL) {
        return -1;
    }
    pvec = create_vector(int);
    if (pvec == NULL) {
        deque_destroy(pdeq);
        return -1;
    }

    vector_init_copy_array(pvec, an_array, 10);
    deque_init_copy_range(pdeq, iterator_next_n(vector_begin(pvec), 3), iterator_prev(vector_end(pvec)));
    printf("The deque: ");
    for (i = 0; i < deque_size(pdeq); ++i) {
        printf("%d ", *(int*)deque_at(pdeq, i));
    }
    printf("\n");

    deque_destroy(pdeq);
    vector_destroy(pvec);

    return 0;
}
  • 结果输出

The deque: 909 111 -174 1233 0 65
