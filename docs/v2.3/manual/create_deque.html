2.2.4. create_deque

  • 函数描述

创建一个双端队列容器。

  • 函数原型
deque_t* create_deque(type);
  • 参数说明
type: 数据类型描述。
  • 备注

type必须是有效的数据类型描述，否则创建失败。 创建成功返回指向deque_t类型的指针，失败返回NULL。

  • 前提条件

包含头文件<cstl/cdeque.h>

  • 示例代码

请参考其他deque_t其他操作函数。
