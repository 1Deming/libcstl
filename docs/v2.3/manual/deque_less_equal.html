2.2.33. deque_less_equal

  • 函数描述

测试第一个双端队列容器是否小于等于第二个。

  • 函数原型
bool_t deque_less_equal(const deque_t* cpdeq_first, const deque_t* cpdeq_second);
  • 参数说明
cpdeq_first: 第一个双端队列容器。
cpdeq_second: 第二个双端队列容器。
  • 备注

如果cpdeq_first等于小于cpdeq_second，返回true，否则返回false。 cpdeq_first和cpdeq_second必须有效的双端队列容器，否则函数的行为是未定义的。 cpdeq_first和cpdeq_second保存的数据类型必须相同，否则函数的行为是未定义的。

  • 前提条件

包含头文件<cstl/cdeque.h>

  • 示例代码
/*
 * deque_less_equal.c
 */

#include <stdio.h>
#include <cstl/cdeque.h>

int main(int argc, char* argv[])
{
    deque_t* pdeq_q1 = NULL;
    deque_t* pdeq_q2 = NULL;

    if ((pdeq_q1 = create_deque(int)) == NULL) {
        return -1;
    }
    if ((pdeq_q2 = create_deque(int)) == NULL) {
        deque_destroy(pdeq_q1);
        return -1;
    }

    deque_init(pdeq_q1);
    deque_init(pdeq_q2);

    deque_push_back(pdeq_q1, 1);
    deque_push_back(pdeq_q1, 2);

    deque_push_back(pdeq_q2, 1);
    deque_push_back(pdeq_q2, 2);
    deque_push_back(pdeq_q2, 4);

    if (deque_less_equal(pdeq_q1, pdeq_q2)) {
        printf("Deque q1 is less than or equal to deque q2.\n");
    } else {
        printf("Deque q1 is greater than deque q2.\n");
    }

    deque_push_back(pdeq_q1, 4);
    if (deque_less_equal(pdeq_q1, pdeq_q2)) {
        printf("Deque q1 is less than or equal to deque q2.\n");
    } else {
        printf("Deque q1 is greater than deque q2.\n");
    }

    deque_push_back(pdeq_q1, 0);
    if (deque_less_equal(pdeq_q1, pdeq_q2)) {
        printf("Deque q1 is less than or equal to deque q2.\n");
    } else {
        printf("Deque q1 is greater than deque q2.\n");
    }

    deque_destroy(pdeq_q1);
    deque_destroy(pdeq_q2);

    return 0;
}
  • 结果输出

Deque q1 is less than or equal to deque q2.
Deque q1 is less than or equal to deque q2.
Deque q1 is greater than deque q2.
