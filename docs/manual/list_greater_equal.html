list_greater_equal

类型描述

测试第一个双向链表容器是否大于等于第二个双向链表容器。

函数原型

bool_t list_greater_equal(const list_t* cplist_first, const list_t* cplist_second);

参数说明

cplist_first: 第一个双向链表容器。
cplist_second: 第二个双向链表容器。

备注

两个双向链表容器必须已经初始化，否则函数的行为未定义。 两个双向链表容器中的数据元素类型必须是相同的，否则函数的行为未定义。 两个双向链表容器的大小由第一个不相等的数据元素的大小关系决定。 如果两个双向链表容器中的数据元素都相等，数据个数也相同，则函数返回true，但是数据个数不同，数据多的容器大。

前提条件

包含头文件<cstl/clist.h>。

版本

所有版本

示例代码

/*
 * list_greater.c
 */

#include <stdio.h>
#include <cstl/clist.h>

int main(int argc, char* argv[])
{
    list_t* plist_l1 = NULL;
    list_t* plist_l2 = NULL;

    plist_l1 = create_list(int);
    if (plist_l1 == NULL) {
        return -1;
    }
    plist_l2 = create_list(int);
    if (plist_l2 == NULL) {
        list_destroy(plist_l1);
        return -1;
    }

    list_init(plist_l1);
    list_init(plist_l2);

    list_push_back(plist_l1, 1);
    list_push_back(plist_l1, 3);
    list_push_back(plist_l1, 1);

    list_push_back(plist_l2, 1);
    list_push_back(plist_l2, 2);
    list_push_back(plist_l2, 2);

    if (list_greater_equal(plist_l1, plist_l2)) {
        printf("List l1 is greater than or equal to list l2.\n");
    } else {
        printf("The l1 is not greater than or equal to list l2.\n");
    }

    list_clear(plist_l1);
    list_clear(plist_l2);

    list_push_back(plist_l1, 1);
    list_push_back(plist_l1, 2);

    list_push_back(plist_l2, 1);
    list_push_back(plist_l2, 2);

    if (list_greater_equal(plist_l1, plist_l2)) {
        printf("List l1 is greater than or equal to list l2.\n");
    } else {
        printf("The l1 is not greater than or equal to list l2.\n");
    }

    list_destroy(plist_l1);
    list_destroy(plist_l2);

    return 0;
}

输出结果

List l1 is greater than or equal to list l2.
List l1 is greater than or equal to list l2.
