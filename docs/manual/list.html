2.3.1. 简介

  • 类型定义
表2-3-1 双向链表list_t容器类型定义
类型定义 简要描述
list_t 双向链表容器类型
list_iterator_t 双向链表容器对应的迭代器类型
  • 操作函数
表2-3-2 双向链表list_t容器操作函数
操作函数 简要描述
create_list 创建一个双向链表容器
list_assign 给双向链表容器赋值
list_assign_array 使用指定的数组给双向链表容器赋值
list_assign_elem 使用指定数据给双向链表容器赋值
list_assign_range 使用指定范围的数据给双向链表容器赋值
list_back 访问双向链表容器的最后一个数据
list_begin 返回指向双向链表容器中的第一个数据的迭代器
list_clear 删除双向链表容器中的所有数据
list_destroy 销毁双向链表容器
list_empty 测试双向链表容器是否为空
list_end 返回指向双向链表容器末尾的迭代器
list_equal 测试两个双向链表容器是否相等
list_erase 删除双向链表容器中指定位置的数据
list_erase_range 删除双向链表容器中指定范围的数据
list_front 访问双向链表容器中的第一个数据
list_greater 测试第一个双向链表容器是否大于第二个双向链表容器
list_greater_equal 测试第一个双向链表容器是否大于等于第二个双向链表容器
list_init 初始化一个空的双向链表容器
list_init_copy 双向链表容器拷贝初始化函数
list_init_copy_array 使用数组初始化双向链表容器
list_init_copy_range 使用指定范围初始化双向链表容器
list_init_elem 使用指定数据初始化双向链表容器
list_init_n 使用指定个数的默认数据初始化双向链表容器
list_insert 在指定的位置插入数据
list_insert_array 在指定的位置插入数组中的数据
list_insert_range 在指定的位置插入指定区间的数据
list_insert_n 在指定的位置插入多个数据
list_less 测试第一个双向链表容器是否小于第二个双向链表容器
list_less_equal 测试第一个双向链表容器是否小于等于第二个双向链表容器
list_max_size 返回双向链表容器的最大可能长度
list_merge 合并两个有序的双向链表容器
list_merge_if 按照指定的规则合并两个有序的双向链表容器
list_not_equal 测试两个双向链表容器是否不等
list_pop_back 删除双向链表容器的最后一个数据
list_pop_front 删除双向链表容器的第一个数据
list_push_back 在双向链表容器的末尾添加一个数据
list_push_front 在双向链表容器的开头添加一个数据
list_remove 删除双向链表容器中指定的数据
list_remove_if 删除双向链表容器中符合指定规则的数据
list_resize 指定双向链表容器的长度，使用默认数据填充
list_resize_elem 指定双向链表容器的长度，使用指定数据填充
list_reverse 把双向链表容器中的数据逆序
list_size 返回双向链表容器中数据的个数
list_sort 排序双向链表容器中的数据
list_sort_if 按照指定规则排序双向链表容器中的数据
list_splice 将源双向链表容器中的数据转移到目的双向链表容器的指定位置
list_splice_pos 将源双向链表容器中指定位置的数据转移到目的双向链表容器的指定位置
list_splice_range 将源双向链表容器中指定范围的数据转移到目的双向链表容器的指定位置
list_swap 交换两个双向链表容器中的数据
list_unique 删除双向链表容器中相邻的重复数据
list_unique_if 删除双向链表容器中符合指定规则的重复数据
