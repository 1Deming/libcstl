2.2.19. deque_front

  • 函数描述

访问双端队列容器中的第一个数据。

  • 函数原型
void* deque_front(const deque_t* cpdeq_deque);
  • 参数说明
cpdeq_deque: 双端队列容器。
  • 备注

函数返回指向容器中第一个数据的指针。 cpdeq_deque必须是有效的双端队列容器，并且已经初始化，否则函数的行为是未定义的。 cpdeq_deque不能为空，否则函数的行为是未定义的。

  • 前提条件

包含头文件<cstl/cdeque.h>

  • 示例代码
/*
 * deque_front.c
 */

#include <stdio.h>
#include <cstl/cdeque.h>

int main(int argc, char* argv[])
{
    deque_t* pdeq = create_deque(int);
    int*     pn_i = NULL;
    int*     pn_j = NULL;

    if (pdeq == NULL) {
        return -1;
    }

    deque_init(pdeq);
    deque_push_back(pdeq, 10);
    deque_push_back(pdeq, 11);

    pn_i = (int*)deque_front(pdeq);
    pn_j = (int*)deque_front(pdeq);
    printf("The first integer of deq is %d\n", *pn_i);
    (*pn_i)--;
    printf("The modified first integer of deq is %d\n", *pn_j);

    deque_destroy(pdeq);

    return 0;
}
  • 结果输出

The first integer of deq is 10
The modified first integer of deq is 9
