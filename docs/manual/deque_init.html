2.2.22. deque_init

  • 函数描述

初始化一个空的双端队列容器。

  • 函数原型
void deque_init(deque_t* pdeq_deque);
  • 参数说明
pdeq_deque: 双端队列容器。
  • 备注

pdeq_deque必须是未初始化的双端队列容器，否则函数的行为是未定义的。

  • 前提条件

包含头文件<cstl/cdeque.h>

  • 示例代码
/*
 * deque_init.c
 */

#include <stdio.h>
#include <cstl/cdeque.h>

int main(int argc, char* argv[])
{
    deque_t* pdeq = create_deque(int);

    if (pdeq == NULL) {
        return -1;
    }

    deque_init(pdeq);
    if (deque_empty(pdeq)) {
        printf("The pdeq is empty.\n");
    } else {
        printf("The pdeq is not empty.\n");
    }

    deque_destroy(pdeq);

    return 0;
}
  • 结果输出

The pdeq is empty.
