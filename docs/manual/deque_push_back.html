2.2.38. deque_push_back

  • 函数描述

向双端队列容器末尾添加一个数据。

  • 函数原型
void deque_push_back(deque_t* pdeq_deque, elem);
  • 参数说明
pdeq_deque: 双端队列容器。
elem: 添加的数据。
  • 备注

pdeq_deque必须有效的双端队列容器，否则函数行为未定义。 elem的类型必须和pdeq_deque中数据的类型一致，否则函数的行为是未定义的。

  • 前提条件

包含头文件<cstl/cdeque.h>

  • 示例代码
/*
 * deque_push_back.c
 */

#include <stdio.h>
#include <cstl/cdeque.h>

int main(int argc, char* argv[])
{
    deque_t* pdeq = create_deque(int);

    if (pdeq == NULL) {
        return -1;
    }

    deque_init(pdeq);

    deque_push_back(pdeq, 1);
    if (deque_size(pdeq) != 0) {
        printf("Last element: %d\n", *(int*)deque_back(pdeq));
    }

    deque_push_back(pdeq, 2);
    if (deque_size(pdeq) != 0) {
        printf("New last element: %d\n", *(int*)deque_back(pdeq));
    }

    deque_destroy(pdeq);

    return 0;
}
  • 结果输出

Last element: 1
New last element: 2
