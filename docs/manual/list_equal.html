list_equal

类型描述

测试两个双向链表容器是否相等。

函数原型

bool_t list_equal(const list_t* cplist_first, const list_t* cplist_second);

参数说明

cplist_first: 第一个双向链表容器。
cplist_second: 第二个双向链表容器。

备注

两个双向链表容器必须已经初始化，否则函数的行为未定义。 两个双向链表容器的数据元素类型必须相同，否则函数的行为是未定义的。 两个双向链表容器中每个数据都对应相等，函数返回true，否则返回fasle。 两个双向链表容器中的数据个数不同，返回false。

前提条件

包含头文件<cstl/clist.h>。

版本

所有版本

示例代码

/*
 * list_equal.c
 */

#include <stdio.h>
#include <cstl/clist.h>

int main(int argc, char* argv[])
{
    list_t* plist_l1 = NULL;
    list_t* plist_l2 = NULL;

    plist_l1 = create_list(int);
    if (plist_l1 == NULL) {
        return -1;
    }

    plist_l2 = create_list(int);
    if (plist_l2 == NULL) {
        list_destroy(plist_l1);
        return -1;
    }

    list_init(plist_l1);
    list_init(plist_l2);

    list_push_back(plist_l1, 1);
    list_push_back(plist_l2, 1);

    if (list_equal(plist_l1, plist_l2)) {
        printf("The lists are equal.\n");
    } else {
        printf("The lists are not equal.\n");
    }

    list_push_back(plist_l2, 2);
    if (list_equal(plist_l1, plist_l2)) {
        printf("The lists are equal.\n");
    } else {
        printf("The lists are not equal.\n");
    }

    list_destroy(plist_l1);
    list_destroy(plist_l2);

    return 0;
}

输出结果

The lists are equal.
The lists are not equal.
